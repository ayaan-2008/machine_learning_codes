# 🌸 IRIS Machine Learning Project

A beginner-friendly Machine Learning project built using Python and Scikit-learn that predicts flower species from the famous Iris dataset.

---

## 📌 Project Overview

This project uses the **Iris Dataset** to train a Machine Learning model capable of classifying flowers into three species:

- Setosa
- Versicolor
- Virginica

The model is trained using supervised learning techniques and demonstrates the complete ML workflow:
- Data Loading
- Data Preprocessing
- Model Training
- Prediction
- Accuracy Evaluation

---

## 🛠 Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib (optional)
- Seaborn (optional)
